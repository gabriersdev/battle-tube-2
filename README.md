# Battle Tube 2

![Banner do projeto](/public/banner.png)

Projeto rodando em: <https://2025.battle.lts.app.br> 

## Descrição

Este projeto é uma aplicação web desenvolvida para apresentar uma "tier list" dos melhores clipes de transmissões ao vivo de 2025. A aplicação permite aos usuários visualizar estatísticas detalhadas (Wrapped) e interagir com a classificação dos clipes.

## Funcionalidades

- Wrapped: Visualização de dados e estatísticas consolidadas dos clipes do ano, com animações, player de música e interação.
- Tier List: Interface para classificação e ranking dos clipes.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias principais:

- Framework: Next.js 16
- Biblioteca UI: React 19
- Linguagem: TypeScript
- Estilização: Tailwind CSS e React Bootstrap
- Animações: Framer Motion
- Utilitários: html2canvas, moment

## Como Executar

### Pré-requisitos

- Node.js
- pnpm

### Instalação

Instale as dependências do projeto:

```bash
pnpm install
```

### Desenvolvimento

Para iniciar o servidor de desenvolvimento:

```bash
pnpm dev
```

A aplicação estará disponível em `http://localhost:3000`.

### Build

Para gerar a versão de produção:

```bash
pnpm build
```

Para iniciar a versão de produção:

```bash
pnpm start
```
